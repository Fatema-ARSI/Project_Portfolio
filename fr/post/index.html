Projet: Optimisation d'un potefeuille financier

L’optimisation d’un portefeuille financier est le processus de sélection du meilleur portefeuille (répartition des actifs), parmi l’ensemble de tous les portefeuilles considérés, selon un objectif definit. Cela afin de maximiser les facteurs tels que le rendement attendu et minimiser risque financier. -Wikipédia Dans ce projet, j’ai extrait les doonnées boursières à l’aide du package yfinance en Python ensuite, j’ai essayé d’optimiser le portefeuille pour avoir un retour sur investissement maximum avec le moins de risque à l’aide d’un package appelé pyportfolioopt créé par Robert Ansrew Martin.
